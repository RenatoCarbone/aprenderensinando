# 📖 Guia de Uso do Sistema - Aprender Ensinando

Bem-vinda ao seu sistema de gestão psicopedagógica! Este guia vai te ensinar a usar todas as funcionalidades.

---

## 🏠 DASHBOARD (Página Inicial)

Após fazer login, você verá o dashboard com:

### Estatísticas Rápidas
- **Total de Pacientes:** Quantos pacientes você tem cadastrados
- **Atendimentos Hoje:** Agendamentos para hoje
- **Atendimentos Mês:** Total de atendimentos do mês
- **Receita Mensal:** Quanto você faturou no mês

### Próximos Atendimentos
Lista dos próximos atendimentos agendados

### Ações Rápidas
Botões para acessar rapidamente as principais funções

---

## 👶 CADASTRAR NOVO PACIENTE

### Quando Usar
Sempre que um novo paciente chegar para a primeira consulta/avaliação

### Como Cadastrar

1. Clique em **"Novo Paciente"** no menu ou no dashboard
2. Preencha o formulário em 3 seções:

#### 1. Dados da Criança
- Nome completo
- Data de nascimento (calcula idade automaticamente)
- Sexo
- Nível escolar, série/ano, escola
- Como conheceu (indicação)

#### 2. Dados dos Responsáveis
- Nomes do pai, mãe, avós ou outros responsáveis
- Responsável principal (quem vai pagar/acompanhar)
- E-mail, telefone/WhatsApp, endereço

#### 3. Entrevista Inicial
**Perguntas importantes para fazer na primeira consulta:**

- **Queixa livre:** "Em que posso ajudá-los? O que os trouxe até aqui?"
- **Quando começou:** "Quando começou o problema?"
- **Como se sentem:** "Como vocês se sentem diante dessa dificuldade?"
- **O que a escola disse:** "O que a escola já lhes disse sobre essa dificuldade?"
- **Observações gerais:** Qualquer outra informação relevante

### Após Cadastrar
- O sistema gera automaticamente um ID (ex: ALU-001, ALU-002...)
- Você pode agendar a primeira aula imediatamente

## 👥 LISTA DE PACIENTES

### Ver Todos os Pacientes

1. Menu → **Pacientes**
2. Você verá uma tabela com todos os pacientes
3. Use os filtros para buscar:
   - Por nome
   - Por ID (ALU-XXX)
   - Por status (Ativos / Pendentes)

### Ver Detalhes de um Paciente

1. Clique em **"Ver Detalhes"** ao lado do paciente
2. Você verá:
   - Todos os dados cadastrais
   - Entrevista inicial completa
   - Histórico de sessões realizadas

### Ações Disponíveis
- **Agendar Aula:** Criar novo agendamento
- **Registrar Sessão:** Anotar uma sessão realizada

---

## 📅 AGENDA DE ATENDIMENTOS

### Agendar Novo Atendimento

1. Menu → **Agenda** → **Novo Agendamento**
2. Selecione:
   - Paciente
   - Data e horário
   - Duração (30min, 1h, etc)
   - Observações (opcional)
3. Clique em **Salvar**

### Visualizar Agendamentos

- **Por data:** Use o filtro de data para ver agendamentos de um dia específico
- **Por paciente:** Busque pelo nome do paciente
- **Resumo:** Veja totais de hoje, semana e mês

### Editar ou Excluir

- **Editar:** Clique no ícone ✏️ ao lado do agendamento
- **Excluir:** Clique no ícone 🗑️

### Status dos Agendamentos
- **Agendado:** Agendamentos futuros (verde)
- **Realizado:** Agendamentos passados (azul)
  - Clique em **"Registrar"** para anotar o que foi feito

---

## 📝 REGISTRAR SESSÃO/ATENDIMENTO

### Quando Usar
Após cada atendimento realizado, para documentar o que aconteceu

### Como Registrar

1. Menu → **Agenda** → Clique em **"Registrar"** no atendimento
   - OU Menu → Lista de Pacientes → Ver Detalhes → **"Registrar Sessão"**

2. Preencha:
   - **Paciente:** Selecionado automaticamente ou escolha
   - **Data da Sessão:** Data do atendimento
   - **Atividades Realizadas:** Descreva o que foi trabalhado
   - **Observações sobre o Aluno:** Comportamento, reações, dificuldades
   - **Progressão/Evolução:** O que melhorou
   - **Próximas Etapas:** Planejamento para próximas sessões

3. Clique em **Salvar Sessão**

### Dicas
- Seja específica nas observações
- Anote comportamentos importantes
- Registre progressos, por menores que sejam
- Use para preparar relatórios futuros

## 💰 CONTROLE FINANCEIRO

### Registrar Receita (Pagamento Recebido)

1. Menu → **Financeiro** → **Novo Lançamento**
2. Preencha:
   - **Tipo:** Receita 💵
   - **Data:** Data do recebimento
   - **Valor:** Quanto você recebeu
   - **Categoria:** Consulta, Avaliação, etc
   - **Descrição:** Ex: "Consulta - Maria Silva"
   - **Paciente:** Selecione se for pagamento de paciente
   - **Observações:** Forma de pagamento, etc

### Registrar Despesa

1. Menu → **Financeiro** → **Novo Lançamento**
2. Preencha:
   - **Tipo:** Despesa 💸
   - **Data, Valor, Categoria, Descrição**
   - Exemplos: Material didático, Aluguel da sala, etc

### Visualizar Resumo

O sistema mostra automaticamente:
- **Receitas do Mês:** Total que você recebeu
- **Despesas do Mês:** Total que você gastou
- **Saldo do Mês:** Lucro (verde) ou prejuízo (vermelho)

### Filtrar Lançamentos

Use os filtros para:
- Ver apenas receitas ou despesas
- Filtrar por mês específico
- Filtrar por categoria

## 💡 DICAS DE USO

### Rotina Diária Recomendada

**🌅 Início do Dia:**
1. Acesse o Dashboard
2. Confira os atendimentos do dia
3. Revise observações dos pacientes agendados

**📝 Após Cada Atendimento:**
1. Registre a sessão enquanto está fresco na memória
2. Anote pontos importantes
3. Se recebeu pagamento, lance no financeiro

**🌙 Fim do Dia:**
1. Revise o que foi feito
2. Verifique se todos os atendimentos foram registrados
3. Planeje o dia seguinte

### 📊 Rotina Mensal

- **Início do mês:** Revise metas e planejamento
- **Durante o mês:** Registre tudo regularmente
- **Fim do mês:** 
  - Analise relatório financeiro
  - Veja evolução dos pacientes
  - Planeje próximo mês

### 🔒 Segurança e Privacidade

- **Nunca compartilhe sua senha**
- **Faça logout ao sair** (botão "Sair" no topo)
- **Dados são privados:** Apenas você vê seus pacientes
- **Backup automático:** Dados salvos no Supabase

## ❓ PERGUNTAS FREQUENTES

**Como editar dados de um paciente?**
> No momento, veja os detalhes do paciente. Para editar, entre em contato com suporte.

**Posso excluir um paciente?**
> Sim, mas isso apagará TODOS os dados dele (sessões, agendamentos, etc). Use com cuidado.

**Como imprimir relatórios?**
> Abra a página desejada e use Ctrl+P. O sistema está otimizado para impressão.

**Posso acessar de tablet/celular?**
> Sim! O sistema é responsivo e funciona em qualquer dispositivo.

**E se eu esquecer minha senha?**
> Na tela de login, clique em "Esqueceu a senha?" e siga as instruções.
